git
init 	해당 폴더에 git 사용 선언
add	파일 추가
reset	add한 파일을 취소
checkout	수정된 파일을 리포짓에 최종적으로 저장된 것과 같은(원래) 상태로 복구
commit	로컬리포지토리(.git폴더)에 내역을 저장
push	commit한 걸 깃허브 리포지토리에 저장
fetch	github에 있는 리포짓을 가져옴
merge	내 폴더의 리포짓과 충돌을 방지하기 위해 사용 


리눅스
ls 	-al 모든 파일을 보여줌
vim 	f1.txt, f1이라는 텍스트를 만들어서 텍스트 편집 화면으로 넘어감
	i를 누르면 텍스트 파일을 편집 가능
	esc를 누르면 읽기모드로 바뀜
	: wq를 입력하면 편집 기능에서 나감

cat	f1.txt, f1 텍스트의 내용을 볼 수 있게 해줌

git commit --amend	커밋 메세지를 bash를 통해 수정할 수 있게 해줌
